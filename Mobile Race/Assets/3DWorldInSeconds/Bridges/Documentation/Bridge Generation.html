Bridge Generation

If the bridge has not been yet generated, the inspector is as follows:

1         Generate Bridge Prefab (not active for prototypes):

By default, the bridge is generated in the scene. If you also want to have a prefab, you have to check this option. If checked, new fields allowing to specify a target prefab folder are displayed (see hereafter).

1.1        Generated Prefab Folder:

This is a read only field showing where the prefab will be stored.

1.2        Specify Prefab Folder:

This button displays a dialog for choosing the folder where you want the generated prefab to be stored.

2         Generate Bridge:

This button builds up the bridge with specified options. If the bridge is built successfully, a message indicates it, otherwise popup error messages are displayed. Once the generation is done, it may be canceled including terrain modifications by the Edit>Undo Generate Bridge (CTRL-Z) command.

 

If the bridge has been generated, the inspector is as FOLLOWS:

3         Delete Bridge:

This button deletes the generated bridge but does not undo any terrain modification. It is recommended if you want to generate a bridge with the same geometry but with other aspects changes such as materials or decorations. If you want to change the geometry, you should instead use the Edit>Undo Generate bridge (CTRL-Z) command.

4         Polygons Count:

This is a report with the number of triangles per LOD (Level of Detail). If there are decorations (statues), the related triangles are detailed. It is also possible to have meshes without Lods for custom decorations. They are reported separately.

5         Lod to keep (This option is not active for prototype bridges):

If you want a bridge without Lods, it is possible to keep only one Lod. This is especially relevant if you want a LOW-POLY bridge. This list allows to specify which Lod you want to keep. To be effective, this choice must be confirmed by the following button.

6         Apply choice:

This button generates a bridge with the chosen Lod. This action may be cancelled by the Edit>Undo Extract Lod (CTRL-Z) command.
