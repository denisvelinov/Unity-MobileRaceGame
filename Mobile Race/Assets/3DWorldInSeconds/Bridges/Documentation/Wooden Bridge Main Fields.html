Wooden Bridge Main Fields

1         Materials Presets (not included for prototypes):

By choosing an option in this list, a combination of materials is assigned to the material slots. If you change a material, the option becomes Custom.

2         Main Material Preview (not included for prototypes):

This is the picture of the texture used for the main material.

3         Main Material (not included for prototypes):

This is the material for the wooden parts of the bridge.

4         Rope Material (not included for prototypes):

This is the material for the rope guardrails.

5         Rope Fastener Material (not included for prototypes):

This is the material for the pieces that fasten the ropes to the wooden ends.

6         Bridge Width:

The width of the bridge.

7         Guardrail Segment:

The guardrail is divided into segments separated by vertical poles. This field specify segments length.

8         Boards Interval:

The width of intervals between floor boards.

9         Guardrails:

There are three options:

         No guardrails,

         Wooden guardrails,

         Rope guardrails

10   Rickety:

Is the bridge rickety? That means that boards length is uneven and boards rotate randomly around several axis.

11   Disorder Factor:

If the bridge is rickety, this field specifies what is the disorder amount.

12   Lacking boards:

If checked, it means that some floor boards are lacking.

13   Lacking Ratio:

This field specifies the average lacking boards ratio. For instance, 0.2 means that, on average, two boards out of ten are lacking.

14   Max Successive Lacking:

This field specifies the maximum number of successive lacking boards.

15   Pillars Interval:

This field specifies the length between two pillars.

16   Crosswise Beams:

If this option is checked, there are horizontal beams between pillars. There are two beams per pillar, a top one and a bottom one.

17   Bottom Beam Height:

This field specifies at which height is the bottom beam. This height may be from the pillars bottom or from the terrain surface (see hereafter).

18   Height is from Ground:

If checked, the Bottom Beam Height is measured from the terrain surface, otherwise it is from the pillars bottom.

19   Lod 0 percentage:

Above this percentage, Lod 0 is displayed. See https://docs.unity3d.com/Manual/LevelOfDetail.html.

20   Lod 1 percentage:

Above this percentage and bellow Lod 0 percentage, Lod 1 is displayed. See https://docs.unity3d.com/Manual/LevelOfDetail.html.

21   Lod 2 percentage:

Above this percentage and bellow Lod 1 percentage, Lod 2 is displayed. Bellow, culling occurs (the game object is no more displayed). See https://docs.unity3d.com/Manual/LevelOfDetail.html.

22   Automatic Pillars Height:

If checked, the bridge pillars height is automatically calculated with the terrain configuration. If unchecked, you have to manually specify the pillars height (see hereafter).

22.1   Manual Pillars Height:

The pillars height is specified manually in meters.
