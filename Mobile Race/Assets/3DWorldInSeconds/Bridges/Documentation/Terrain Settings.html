Terrain Settings

1         Modify Terrain:

If checked, the terrain is modified as the bridge is generated. If several trials are performed at the same location, for instance to change materials, this checkbox should be unchecked after the first generation. Otherwise, successive smoothing process might result in terrain oddities.

2         Remove Trees:

If you want trees to be removed in the bridge vicinity, there are two possible choices:

         The smart option analyses each tree geometry and removes it if it intersects the bridge bounds. The advantage is to keep small trees close to the bridge while removing big ones.

         The By Distance option removes trees if their center are located bellow a specified distance from the bridge axis. There is hence no difference between small trees and big ones. If this option is chosen, a field allowing to write the distance is displayed.

3         Remove Grass:

If checked, the terrain grass (details) is removed as soon as the grass patch distance from the bridge bounds is under a specified value. Please note that this operation accuracy depends on the terrain detail resolution. Lower resolution means that bigger patches are removed at once.

3.1        Details distance from bridge:

This field is shown when the Remove Grass option is checked. It allows to specify at which distance from the bridge bounds, grass patches are removed. The actual removal distance for each plant depends on this value and on the terrain detail resolution because grass is managed by whole squares. A high terrain detail resolution will generate more accurate results, but might decrease overall performances.

4         Level Terrain at Start:

If checked, the terrain elevation is modified at the bridge start in order to get an even surface. As the bridge height is calculated to fit the start point, the terrain elevation is only slightly modified. The terrain elevation is also adjusted along a given distance (see hereafter) to get a smooth transition between the existing terrain and the bridge.

5         Automatic Adjustment:

If checked, the adjustment distance is automatically calculated. If unchecked, a field specifying this distance is displayed for manual input (see hereafter).

5.1        Adjustment Distance:

This field allows to manually specify the adjustment distance. If set to zero, there is no adjustment, possibly resulting in a steep slope.

5.2        Get value Recommendation:

Click this button to get a value recommendation for the adjustment distance. This value is the one that would have been used if the Automatic Adjustment option had been chosen. This value is displayed inside a popup message. If it fits your needs, you may paste it in the Adjustment Distance field.

6         Level Terrain at End:

If checked, the terrain elevation is modified at the bridge end in order to get a surface that fits the bridge height. As this bridge height is calculated to fit the start point, the terrain elevation may be heavily modified if the initial two banks are at very different elevations. The terrain elevation is also adjusted along a given distance (see hereafter) to get a smooth transition between the existing terrain and the bridge. If you want to build a pier rather than a bridge, just uncheck this option.

7         Automatic Adjustment:

If checked, the adjustment distance is automatically calculated. If unchecked, a field specifying this distance is displayed for manual input (see hereafter).

7.1        Adjustment Distance:

This field allows to manually specify the adjustment distance. If set to zero, there is no adjustment, possibly resulting in a steep slope.

7.2        Get value Recommendation:

Click this button to get a value recommendation for the adjustment distance. This value is the one that would have been used if the Automatic Adjustment option has been chosen. This value is displayed inside a popup message. If it fits your needs, you may paste it in the Adjustment Distance field.

8         Remove Bank Lip:

If the start terrain is lower than the end terrain, the latter is dug in order to fit the bridge height. As the terrain is only dug after the bridge, it may occur that parts of terrain along the bridge sides are higher and produce a kind of lip. If you want to get rid of this lip, you have to check this option, then a field specifying the Lip Removal Distance is displayed for manual input (see hereafter).

8.1        Lip Removal Distance:

This field allows to manually specify the distance along the bridge sides where the terrain should be lowered in order to be at the same height as terrain at the bridge end.

8.2        Get value Recommendation:

Click this button to get a value recommendation for the Lip Removal distance. This value is displayed inside a popup message. If it fits your needs, you may paste it in the Lip Removal Distance field.

9         Smooth Edges:

If the terrain at bridge ends undergoes an important elevation increase, it may lead to a very steep bank. Check this option if you want to get a smoother bank. In this case, extra fields will be displayed in order to specify the smoothing strength (see hereafter).

9.1        Smoothing Strength:

The higher this value is, the smoother the bank will be. For high values, the river width may be significantly reduced.

10   Natural Look Noise:

When the terrain is adjusted along a long distance, the even slope may look unnatural. If you want to get a more natural look, you may specify a noise amount.
